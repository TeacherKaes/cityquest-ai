# Schnellstart für Lehrkräfte

## Ziel

Du brauchst keine technische Bedienung.

## Ablauf

1. Öffne die Web-App.
2. Klicke auf **Quest erstellen**.
3. Wähle:
   - Edition
   - Sprachniveau
   - Dauer
4. Klicke auf **QR-Code erzeugen**.
5. Zeige den QR-Code.
6. Schülerinnen und Schüler scannen und starten.

## Teacher Commands während des Spiels

- Hint geben
- Mission überspringen
- Finale starten
- Sprache vereinfachen
- Safety Mode

Diese Funktionen werden in der nächsten Version als Live-Dashboard ausgebaut.
